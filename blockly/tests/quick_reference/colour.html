Colour

Colour Picker

The colour picker block is the simplest way to get a colour. It appears as a rounded rectangle with the colour you choose within the rectangle. When opened a palette of colours appears, any colour may be chosen. The block will then return that colour.

Random Colour

The random colour block generates colours at random. Specifically, each of the red, green, and blue components is a number from 0 to 255 (inclusive), with equal likelihood.

Colour RGB

The colour rgb block allows you to specify the desired percent of red, green, and blue. A range from 0 to 100 (inclusive) is used for each component. The example block below, builds a colour with the maximum amounts of red and blue and no green (making purple).

Colour Blend

The colour blend block allows for two colours to be blended. This block also requires a ratio between (0.0 and 1.0) to be specified. If you provide a ratio of 0.0 the result would be all of the second colour and none of the first.
