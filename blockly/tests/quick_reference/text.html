Text

Text

The text block can contain letters (lower or upper case), numbers, punctuation marks, other symbols,, and blank spaces.

Create text with

The create text with block combines (concatenates) the value of two or more strings, numbers, or variables. This block also allows you to increase the number of inputs to be combined. The example below simple combines a variable holding a string and a string together. Then the block returns the new combined string. Note that there is no space between them, since there were no spaces in either original text.

Append Text

The append text block adds the given text to the specified variable. In the example below, it changes the value of the variable greeting from "hello" to "hello, there!".

Length Of

The length of block counts the number of letters, numbers, etc.,in each text block. The length of "We're #1" is 9, and the length of the empty text is 0.

Is Empty

The is empty block check whether the given text is empty (has length 0). This block will return either true or false. This first example below of an empty string will return true. The second example below of a variable greeting, equal to "Hello", will return false.
