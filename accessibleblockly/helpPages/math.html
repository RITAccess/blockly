Math

Number

This is a simple number block.

Operations

The operations block has 5 different modes to choose from: add, subtract, multiply, divide, and exponentiation. The block will return the result of the operation.

Math on Number

This block will return the value from the operation you choose. The operation modes are: square root, absolute, negative, ln, log, e to the power, ten to the power.

Trigonometry Functions

This block has the six trigonometry functions. They are: sine, cosine, tangent, co-secant, secant, cotangent.

Constants

This block has 6 common constants that are returned on use. They are: pi, e, phi, square root of 2, square root of one half, infinity.

Type of Number

The type of number block has 7 modes: even, odd, prime, whole, positive, negative, and divisible by. The block will check the number you provide in the block against the mode in use and it will return a boolean value. The divisible by mode is slightly different than others as the block will need 2 numbers. Check out the example below.

Change Variable by

The change variable by key will do just that. Choose a variable name to change and a number to change it by. This block does not return anything but it will set the variable to the new value.

Round

The round block will round any number. There are three modes: round, round up, and round down. Round will use conventional rounding methods, round up will always give a greater number

Math on List

The math on list block will return the sum, min, max, average, median, modes, standard deviation, or a random item from the list you provide.

Modulo

The modulo block will compute the division between the two numbers provided within the block. Then, the remainder will be returned.

Constrain

The constrain block requires a number to be given as well as two values for the number to be constrained between. A low and a high constraint value are required. If the given number is less than the low constraint value then the block will return the low number and the same with the high constraint value. This block is inclusive.

Random Integer

The random integer block will return a random integer between the two specified limits. This block is inclusive.

Random Fraction

The random fraction block will return a random fraction between 0.0 and 1.0.The 0.0 is inclusive and the 1.0 is exclusive.
