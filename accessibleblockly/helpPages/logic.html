Logic

If

The simplest conditional statement is an if block. The below example when run will compare the value of the variable x to one hundred. If x is larger, the body of the if block, the do, will be run and "What a big number!" will be printed. Otherwise, nothing happens.

If-Else

With the if block it is also possible to specify that something should happen if the conditional is not true. The if block has a mutation which allows for an else to be added to the block. This example is similar to the above example where if x is greater than one hundred, "What a big number" will be printed. Otherwise, "That's not very big" will be printed.

If-Else-If

With the if block it is also possible to test multiple conditions with a single if block by adding else if clauses. An if block can have any number of else if sections. Conditions are evaluated top to bottom until one is satisfied, or until no more conditions are left. The example below demonstrates testing multiple conditions. If x is greater than one hundred, "What a big number!" will be printed. If it is not greater than one hundred, it goes on to check if x is equal to forty-two. If so, it prints "That's my lucky number!". Otherwise, nothing happens.

If-Else-If-Else

If blocks may also have both else if and else sections. The else section guarantees that some action is performed, even if none of the prior conditions are true. An else section may occur after any number of else if sections (including zero) but never before an else if.

Comparisons

There are six comparison operators. Each takes two inputs(normally numbers) and returns true or false depending on how the inputs compare with each other. The six operators are: equals, not equals, less than, less than or equal, greater than, greater than or equal.

Logical Operations

The logical block with the and operator will return true if both of its two inputs are also true. The or operator block will return true if either of its two inputs are true.

Not

The not block converts its Boolean input into its opposite. For example, the result of not true is false. See the block example below.

True or False

Boolean algebra is a mathematical system that has two values. True and False. Boolean values are also called conditions.

Null

Null is a non-value. It is a place holder for a data value that is not know or has not been specified.
