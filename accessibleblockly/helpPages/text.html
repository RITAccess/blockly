Text

Text

The text block can contain letters (lower or upper case), numbers, punctuation marks, other symbols, and blank spaces.

Create text with

The create text with block combines (concatenates) the value of two or more strings, numbers, or variables. This block also allows you to increase the number of inputs to be combined. The example below simple combines a variable holding a string and a string together. Then the block returns the new combined string. Note that there is no space between them, since there were no spaces in either original text.

Append Text

The append text block adds the given text to the specified variable. In the example below, it changes the value of the variable greeting from "hello" to "hello, there!".

Length Of

The length of block counts the number of letters, numbers, etc.,in each text block. The length of "We're #1" is 9, and the length of the empty text is 0.

Is Empty

The is empty block check whether the given text is empty (has length 0). This block will return either true or false. This first example below of an empty string will return true. The second example below of a variable greeting, equal to "Hello", will return false.

Finding Text

This block can be used to check whether a piece of text is in another piece of text and, if so, where it appears. There is the option to find either the first or last occurrence of the text. If the piece of text does not contain the other piece of text, the result will be 0. See the example below whose result is 2.

Extract a single character

This block will return the single character being looked for. This block does not modify the text which on which the extraction is performed. The options of which character to extract are: the index, index from the end, first letter, last letter, or a random letter. The example below shows the extraction of the last letter in text "abcde". The result is "e".

Extract a region of text

The in text...get substring block allows a region of text to be extracted, starting with either: index, index from the end, or first letter. Ending with: index, index from the end, or last letter. The example below gets a substring from the first letter to letter index 3. "abc" is extracted for this example.

Adjusting text case

This block creates a version of the input text that is either: UPPER CASE (all letters upper-case), lower case, or Title Case (first letters upper-case, other letters lower-case). Non-althabetic characters are not affected. Note that this block will not work with languages without case such as Chinese. The example below shows the text "hello" (all lower-case) adjusted to all upper-case letters.

Trimming Spaces

The trimming spaces block removes space characters from either: the beginning of the text, the end of the text, or both sides of the text. The example below trims spaces from both sides of the text " hi there ". The result is "hi there" (Note the spaces in the middle of the text are not affected).
